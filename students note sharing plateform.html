How to play
Use Arrow Keys or W A S D to move. Eat the green food to grow. Avoid hitting walls or yourself.

Game Info

Grid 20 x 20 cells

Speed Adjustable in code (default 120ms per tick)

Features Score, Pause, Restart, Smooth rendering

Tip: Press Space to pause/resume.

About

This is a compact, easy-to-read implementation of the classic Snake game using HTML Canvas and vanilla JavaScript.
